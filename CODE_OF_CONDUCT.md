# Code of Conduct

This project follows a simple standard: be respectful, constructive, and inclusive.

## Expected behavior
- Be kind and professional.
- Assume good intent.
- Give actionable feedback.
- Respect differing viewpoints.

## Unacceptable behavior
- Harassment, discrimination, or personal attacks.
- Trolling or deliberately disruptive behavior.
- Sharing private/sensitive information without consent.

## Enforcement
Project maintainers may remove or reject contributions and interactions that violate this code.
