# Contributing

Thanks for your interest in contributing.

## Development setup
1. Create a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Ensure TA-Lib is installed on your system.

## Contribution guidelines
- Keep changes focused and small.
- Add/update documentation for behavioral changes.
- Avoid committing generated datasets under `resources/tickerList/<TICKER>/` and `resources/trades/`.
- Do not commit secrets, tokens, or private credentials.

## Pull requests
Please include:
- What changed
- Why it changed
- Any local validation steps used
