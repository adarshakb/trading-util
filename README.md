# Trading util

An experimental repo for a trading tool

### Requirements
1. https://pypi.org/project/yfinance/
2. https://github.com/mrjbq7/ta-lib/

### Usage
1. Run the `downloadData.py` inside `src/dataUtils/downloadData.py` first to download data into `resources` folder
2. Play around with technical analysis kernals in `src/analysis/ta` folder
